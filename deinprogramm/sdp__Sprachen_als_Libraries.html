5 sdp: Sprachen als Libraries

Note: This is documentation for the language levels that go with the German textbook Schreibe Dein Programm!.

5.1 Schreibe Dein Programm - Anfänger

 (require deinprogramm/sdp/beginner)
  package: deinprogramm

Das Modul deinprogramm/sdp/beginner implementiert die Anfängersprache für Schreibe Dein Programm!; siehe Schreibe Dein Programm! - Anfänger.

5.2 Schreibe Dein Programm!

 (require deinprogramm/sdp) package: deinprogramm

Das Modul deinprogramm/sdp implementiert die Standardsprache für Schreibe Dein Programm!; siehe Schreibe Dein Programm!.

5.3 Schreibe Dein Programm! - fortgeschritten

 (require deinprogramm/sdp/advanced)
  package: deinprogramm

Das Modul deinprogramm/sdp/advanced implementiert die fortgeschittene Sprachebene für Schreibe Dein Programm!; siehe Schreibe Dein Programm! - fortgeschritten.
