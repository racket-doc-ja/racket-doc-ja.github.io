6 Compiling to Scheme

This library implements a JavaScript-to-Scheme compiler. It be can required via:

 (require javascript/compile) package: javascript

procedure

(compile-script elts) → syntax?

  elts : (listof SourceElement?)
Compiles a JavaScript script to Scheme.

procedure

(compile-interaction elts) → syntax?

  elts : (listof SourceElement?)
Compiles a JavaScript REPL interaction to Scheme.

procedure

(compile-module elts) → syntax?

  elts : (listof SourceElement?)
Compiles a JavaScript module to Scheme.

procedure

(compile-expression expr) → syntax?

  expr : Expression/X?
Compiles a JavaScript script to Scheme.
