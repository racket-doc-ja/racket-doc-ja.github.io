8 Evaluation

This library provides facilities for evaluating JavaScript. It can be required via:

 (require javascript/eval) package: javascript

procedure

(eval-script src [ns]) → any

  src : (or/c input-source? syntax?)
  ns : namespace? = (make-js-namespace)
Evaluates a JavaScript script from source src in namespace ns.

procedure

(eval-expression src [ns]) → any

  src : (or/c input-source? syntax?)
  ns : namespace? = (make-js-namespace)
Evaluates a JavaScript expression from source src in namespace ns.
