Nonfree SRFI Specification Documents

 SRFI 5 specification package: srfi-doc-nonfree

This package contains a copy of the SRFI 5 specification document.

For historical reasons, the SRFI 5 specification document has a restrictive license and is not included in the main Racket distribution.

Racket provides a free implementation of SRFI 5 in srfi/5 and free alternative documentation for SRFI 5’s variant of let in SRFI 5: A compatible let form with signatures and rest arguments. The implementation and documentation are distributed under the same license as Racket: only the original specification document is restrictively licensed.
