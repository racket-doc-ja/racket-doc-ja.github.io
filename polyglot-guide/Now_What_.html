7 Now What?

If you’ve read the guide up to here, you’re probably interested in using Polyglot for your website. If that’s true, then thank you. I sincerely hope you have fun with it.

If you want to say thank you in return, please see Contributions.

If you want to learn tips and tricks to supplement your knowledge, see Polyglot: Tutorials and Polyglot: How-Tos.

I actively use Polyglot for my own website and convert any useful notes to tutorials and how-tos. If you see something a feature appear that catches your eye and a how-to is not available, please contact me.
