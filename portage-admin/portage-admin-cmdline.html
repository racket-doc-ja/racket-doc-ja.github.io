1 Console usage

  • -m or --main — show the main window

  • -e or --eselect — show the eselect modules explorer

  • -l or --log — show the log files explorer

  • -p or --pkgdb — show the package database explorer

  • -h or --help — show help information with usage options

  • -V or --version — show program version
