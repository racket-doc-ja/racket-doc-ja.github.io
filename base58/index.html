Marc Burns <marc@kn0x.io>

 (require base58) package: base58

Provides Base58Check encoding and decoding functions.

procedure

(base58-encode bs [#:check? check?]) → string?

  bs : bytes?
  check? : any/c = #t
Encodes bs to a base58 string. When check? is not #f, appends check bytes.

procedure

(base58-decode str [#:check? check?]) → bytes?

  str : string?
  check? : any/c = #t
Decodes base58 string str to bytes. When check? is not #f, validates and removes check bytes.
