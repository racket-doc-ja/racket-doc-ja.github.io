value

into-hash : (reducer/c entry? immutable-hash?)

value

into-mutable-hash : (reducer/c entry? mutable-hash?)

A pair of reducers that build either an immutable hash table or a mutable hash table from a sequence of entries. Duplicate keys are not allowed, and attempting to reduce a sequence containing duplicate keys will raise exn:fail:contract.

Example:
> (for/reducer into-hash
               ([str (immutable-string-split "The quick brown fox")])
    (entry (string-downcase str)
           (immutable-string-length str)))

'#hash(("brown" . 5) ("fox" . 3) ("quick" . 5) ("the" . 3))

(combine-into-hash value-combiner)

procedure

(combine-into-mutable-hash value-combiner)

 → (reducer/c entry? mutable-hash?)
  value-combiner : (-> any/c any/c any/c)
Constructs a reducer that combines a sequence of entries into either an immutable hash table or a mutable hash table, respectively. Values for duplicate keys are combined using value-combiner.

Example:
> (reduce (combine-into-hash immutable-string-append)
          (entry 1 "apple")
          (entry 1 ".")
          (entry 1 "banana")
          (entry 2 "orange")
          (entry 2 ".")
          (entry 2 "grape"))

'#hash((1 . "apple.banana") (2 . "orange.grape"))

)

empty-hash : empty-immutable-hash? = (hash)

The empty (immutable) hash table.

procedure

(hash-ref-safe h k) → option?

  h : immutable-hash?
  k : any/c
Returns an option containing the value mapped by k in h, returning absent if no such value exists.

Examples:
> (hash-ref-safe (hash 'a 42) 'a)

(present 42)

> (hash-ref-safe (hash 'a 42) 'b)

#<absent>
