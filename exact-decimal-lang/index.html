exact-decimal-lang

source code: https://github.com/AlexKnauth/exact-decimal-lang

 #lang exact-decimal package: exact-decimal-lang
The exact-decimal language is a lang-extension like at-exp that reads decimals as exact rationals.
For example, #lang exact-decimal racket/base is a language like racket/base, except that for example 0.1 reads as 1/10. You can still specify inexact numbers with #i though.

#lang exact-decimal racket/base
0.1
produces 1/10
