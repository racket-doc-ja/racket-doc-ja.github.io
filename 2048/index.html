2048

Philip McGrath <philip@philipmcgrath.com>

 (require 2048/main/gui/2048) package: 2048

This package provides a Racket version of the game 2048. It currently works as a GUI application, with a launcher installed by raco setup. Support for a text-based interface is planned in the future.

The game is inspired by Gabriele Cirulli’s JavaScript version: the implementations don’t share any code, but, for example, the values of certain constants are the same as in Cirulli’s version, so the difficulty level should be comparable.
