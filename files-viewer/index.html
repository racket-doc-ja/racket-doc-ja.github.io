Files-viewer: a file manager plugin for DrRacket

风变科技-矩阵工作室 Qi Jiang <1227847322 at qq dot com>

1 Introduction

Files-viewer is a file manager plugin for DrRacket.

2 Installation

To install, either look for files-viewer in the DrRacket menu File > Package Manager, or run the raco command:
  raco pkg install files-viewer

You need to restart DrRacket. Now you should have a new item Show the File Manager in the View menu.

3 Usage

Click View then Show the File Manager.

To edit a file, double click the item.

To call the popup menu, right click the file manager.

To show or hide the file manager, View > Show/Hide the File Manager.
